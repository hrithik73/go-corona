## Running on Local Machine
1. Clone the repository to your machine

   ```bash
   git clone https://github.com/hrithik73/go-corona.git
   cd go-corona
   ```

2. Install packages

   If you use `npm`

   ```bash
   npm install
   ```

   or

   If you use `yarn`

   ```bash
   yarn
   ```

3. Run the development server

   If you use `npm`

   ```bash
   npm start
   ```

   or

   If you use `yarn`

   ```bash
   yarn start
   ```

4. Visit <http://localhost:3000>
